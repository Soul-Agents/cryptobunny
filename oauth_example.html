Twitter OAuth Example

This example demonstrates how to connect a Twitter account using the OAuth flow.

Not authenticated. Click "Connect Twitter Account" to start.

Post a Tweet

How it works

  1. We start by calling the /connect-twitter-account endpoint
  2. The endpoint gives us an auth_url to redirect to Twitter
  3. Twitter handles the authentication and redirects to our backend
  4. Our backend processes the callback and redirects to our frontend
  5. We can check authentication status using the client_id
  6. Once authenticated, we can post tweets using the /post-tweet endpoint
